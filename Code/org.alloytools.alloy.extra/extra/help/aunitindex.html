Welcome to the
AUnit Tutorial

This tutorial will outline key concepts of AUnit, a testing framework for Alloy, as well as how to use the built in support features within this version of the Alloy Analyzer.

This user guide is divided into the following sections:
     0. Motivation: Outlines the design choices of and motivation for AUnit.
     1. Defining a Test Case: Outlines what an AUnit test case is.
     2. What does it mean to execute an AUnit test: Outlines what makes a passing or failing test case.
     3. How to add an AUnit test to a model?: Outlines how to use the new Alloy Analyzer syntax to create AUnit test cases within the Analyzer.
     4. What is model coverage in Alloy?: Outlines what coverage means for Alloy and how coverage is calculate for any given AUnit test case.
